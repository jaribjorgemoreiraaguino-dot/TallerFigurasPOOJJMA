Taller Figuras POO

Implementación del taller: clases para figuras geométricas en Python.

Archivos incluidos
- figura_geometrica.py: clase base con validaciones y métodos comunes.
- cuadrado.py: clases que implementan área y perímetro para un cuadrado.
- rectangulo.py: implementa área y perímetro para rectángulo.
- main.py: programa principal que crea objetos, muestra resultados y prueba validaciones.

Ejecución
Ejecutar `python main.py` desde la carpeta donde se encuentran los archivos.

Observaciones
- Se han incluido validaciones en los setters (alto y ancho > 0).
- La clase base define `perimetro()` como método a sobrescribir por las subclases.
